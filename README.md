# 🎮 TSI Runner

Um jogo simples desenvolvido com **HTML**, **CSS** e **JavaScript**, originalmente feito em sala de aula e posteriormente adaptado para incluir um **nível de dificuldade mais desafiador**.

---

##  Sobre o Projeto

O **TSI Runner** é um jogo de corrida onde o jogador deve evitar obstáculos e alcançar a maior pontuação possível.
Foi criado como parte de um exercício acadêmico para praticar lógica de programação, manipulação de DOM e eventos de teclado.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** → estrutura do jogo
* **CSS3** → estilo e design responsivo
* **JavaScript (puro)** → lógica, animações e pontuação

---

##  Funcionalidades

* Sistema de pontuação dinâmica
* Dificuldade progressiva conforme o jogador avança
* Reinício automático após colisão
* Efeitos visuais e sonoros simples

---

##  Objetivo

Demonstrar habilidades básicas de desenvolvimento web front-end e manipulação de elementos dinâmicos na tela, além de aplicar conceitos de jogos 2D simples no navegador.
